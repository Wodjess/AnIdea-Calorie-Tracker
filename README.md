# AnIdea — Calorie & Nutrition Tracker

> Android app for tracking daily calorie intake, macronutrients (proteins, fats, carbs), and managing a personal food diary.

## Features
- Track daily calories, proteins, fats, and carbohydrates
- Add custom foods with nutritional data
- Gram calculator for portion sizing
- Food diary with daily totals
- User profile management

## Tech Stack
- **Language:** Kotlin
- **Platform:** Android (SDK)
- **UI:** ViewBinding, RecyclerView
- **Architecture:** Activity-based with ViewBinding

## Screens
- **Main** — Food list with nutritional breakdown
- **Add Food** — Custom food entry form
- **Calculator** — Portion gram calculator
- **Profile** — User settings and daily goals
